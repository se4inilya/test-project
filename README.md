## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```

## Config

```bash
# db postgres connection settings are in .env file, change them accordingly to your db creds

# to run app just enter
$ npm install
# then
$ npm run start:dev
```
